# Prompt 03 — Traduzioni i18n (it / en)

## Contesto

I file di traduzione si trovano in:

- `src/main/webapp/i18n/it/` (Italiano)
- `src/main/webapp/i18n/en/` (Inglese)

Prendere come riferimento il file `src/main/webapp/i18n/it/pagoPaPaymentReceipt.json`
per la struttura JSON e le chiavi radice.

Anche il file `src/main/webapp/i18n/it/global.json` contiene la voce di menu
che dovrà essere aggiornata per la nuova sezione.

## Aggiornare `src/main/webapp/i18n/it/global.json`

Aggiungere nella sezione `global.menu` la voce di menu per la nuova sezione.
Aprire il file e, all'interno dell'oggetto `"registry"`, aggiungere:

```json
"ricercaOperazioni": "Ricerca Operazioni"
```

## Aggiornare `src/main/webapp/i18n/en/global.json`

Aprire il file e aggiungere nella stessa posizione:

```json
"ricercaOperazioni": "Operation Search"
```

## ⚠️ Note

- Adattare le etichette alle linee guida UX del progetto se necessario.
- I valori presenti nei file JSON dei dettagli token ed eventi dovranno essere integrati
  una volta definiti i campi reali restituiti dalle API.
